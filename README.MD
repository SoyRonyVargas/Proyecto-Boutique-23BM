## Arrancar el proyecto

Restaura los paquetes desde la consola con
```sh
dotnet restore
```

Ejecuta las migraciones en la terminal con
```sh
dotnet ef database update
```

O con la terminal de paquetes
```sh
Update-Database
```

Dale al boton de play en visual studio.
